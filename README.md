# Track SP

Web tinh gom 2 chuc nang rieng:

- Luu link va mo nhanh.
- Dang ky tung thiet bi dien thoai truoc, khi can dung thi bam bat dau de dem 24 gio.
- Co the nhap gio bat dau trong ngay khi luu may, hoac nhap gio rieng tren tung may truoc khi bam bat dau 24h.

## Chay local

Mo truc tiep file `index.html` bang Chrome.

Hoac chay trong PowerShell:

```powershell
start .\index.html
```

## Cau hinh Supabase

Neu chua cau hinh du bang Supabase, app van fallback ve `localStorage` tren trinh duyet.

### Bang links

```sql
create table public.links (
  id bigint generated by default as identity primary key,
  url text not null,
  created_at timestamptz not null default now()
);

alter table public.links enable row level security;

create policy "Allow public read links"
on public.links for select
using (true);

create policy "Allow public insert links"
on public.links for insert
with check (true);

create policy "Allow public delete links"
on public.links for delete
using (true);
```

### Bang devices

```sql
create table public.devices (
  id bigint generated by default as identity primary key,
  name text not null,
  registered_at timestamptz not null default now(),
  started_at timestamptz
);

alter table public.devices enable row level security;

create policy "Allow public read devices"
on public.devices for select
using (true);

create policy "Allow public insert devices"
on public.devices for insert
with check (true);

create policy "Allow public update devices"
on public.devices for update
using (true)
with check (true);

create policy "Allow public delete devices"
on public.devices for delete
using (true);
```

Neu da tao bang `devices` tu ban cu, chay them:

```sql
alter table public.devices
add column if not exists started_at timestamptz;
```

Sau do sua `config.js`:

```js
window.TRACK_SP_CONFIG = {
  SUPABASE_URL: 'https://YOUR_PROJECT.supabase.co',
  SUPABASE_ANON_KEY: 'YOUR_ANON_KEY'
};
```
