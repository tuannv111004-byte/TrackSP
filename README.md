# Track SP Links

Web tinh de dan link, luu bang `localStorage`, va bam mo nhanh. Code dung HTML/CSS/JavaScript co ban de chay tot tren Chrome cu.

## Chay local

Mo truc tiep file `index.html` bang Chrome.

## Deploy Vercel

Project nay la static site. Co the import folder/repo nay vao Vercel, hoac dung Vercel CLI:

```bash
vercel --prod
```
